# Koopa

Library for generating/parsing/optimizing Koopa IR.

For more details, please see the readme file of [the koopa repository](https://github.com/pku-minic/koopa), and [the documentation](https://docs.rs/koopa).

## Usage

```
cargo add koopa
```
